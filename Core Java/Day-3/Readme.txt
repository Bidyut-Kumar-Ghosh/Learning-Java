How to Run the Programs
To run any of the Java programs, follow these steps:

1. Open your terminal or command prompt.
2. Navigate to the directory containing the Java file you want to run.
3. Compile the Java file using the following command:
   ```bash
   javac FileName.java
   ```
4. Run the compiled program with the command:
   ```bash
   java FileName
   ```

Replace `FileName` with the name of the Java file you want to execute (without the `.java` extension).
